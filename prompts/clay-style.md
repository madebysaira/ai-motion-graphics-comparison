# Clay Style

Playful claymation aesthetic for motion graphics.

## Style Block

```
Style: Claymation, playful, tactile
Colors: Warm pastels — soft pink (#f8bbd9), mint (#a5d6a7), sky blue (#90caf9), cream (#fff9c4)
Font style: Rounded, friendly, hand-drawn feel
Elements: Clay-textured objects, soft shadows, organic shapes
Background: Solid pastel or subtle gradient
Typography: Rounded, friendly, slightly irregular (hand-crafted feel)
```

## Action Block Example

```
Action: Clay character waves hello, bounces slightly with squash-and-stretch
Camera: Static medium shot
Duration: 4 seconds
```
