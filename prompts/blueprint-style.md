# Blueprint Style

Technical blueprint aesthetic for motion graphics.

## Style Block

```
Style: Technical blueprint, architectural
Colors: Blueprint blue (#1e3a8a), white lines (#ffffff), accent yellow (#fbbf24)
Font style: Monospace or technical sans-serif, all caps
Elements: Grid lines, measurement markers, technical diagrams, wireframe objects
Background: Blueprint blue with subtle grid pattern
Typography: White, monospace, technical labels
```

## Action Block Example

```
Action: Wireframe smartphone rotates 360 degrees with measurement lines and dimension labels
Camera: Static with object rotation
Duration: 6 seconds
```
