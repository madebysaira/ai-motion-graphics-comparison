# Contributing to AI Motion Graphics Comparison

Thanks for considering a contribution! This repo is meant to evolve with real-world testing from working creators.

## What I'm Looking For

### High Priority

- **Model comparisons** with specific use cases (e.g., "Tested X model for Y type of motion graphics")
- **Prompt templates** that worked on client work (with style sheets, action blocks)
- **Hybrid workflows** (Omni + Remotion + After Effects pipelines)
- **Cost/performance benchmarks** (actual generation counts, success rates, time budgets)
- **Workarounds** for known limitations (face blocking, text artifacts, logo handling)

### Also Welcome

- Bug fixes in documentation
- Additional style examples (glass, paper cutout, 3D render, etc.)
- Translation of docs to other languages
- Video examples (links to YouTube/Vimeo with breakdown)

## How to Contribute

### 1. Test and Document

1. Use the model(s) on real work or test projects
2. Document what worked, what failed, how many generations needed
3. Save prompt templates that produced good results
4. Note any workarounds you discovered

### 2. Submit Pull Request

**PR Description should include**:
- What problem this solves
- Which model(s) tested
- Success rate / generation count
- Example output (link or description)
- Any prerequisites (specific platform, credits, etc.)

### 3. Review Process

I'll review for:
- ✅ Works as described
- ✅ No hardcoded secrets or platform-specific URLs (unless noted)
- ✅ Follows existing format (style block + action block)
- ✅ Documentation clear and actionable

Expect feedback within a week.

## Format Guidelines

### Prompt Templates

Use the established format:
```
[STYLE - LOCKED]
Style: [description]
Colors: [specific values]
Font style: [description]
Elements: [list]
Background: [description]
Typography: [notes]

[ACTION]
Action: [specific motion]
Camera: [move or static]
Duration: [seconds]
```

### Model Comparisons

Include:
- **Best for**: Primary use case
- **Strengths**: 3-5 bullets
- **Weaknesses**: 3-5 bullets
- **Usability**: X% for commercial work
- **When I use it**: Specific scenarios
